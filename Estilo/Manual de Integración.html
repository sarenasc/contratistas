Archivos del proyecto

Copia estos 4 archivos a tu carpeta de assets.

Estructura
# Copiar a app/assets/ o public/css|js/

tu-proyecto/
├── styles.css        ← Tokens + layout + componentes base
├── animations.css    ← Keyframes, fade, stagger, skeleton
├── components.css    ← Inputs, selects, progress, forms
└── multiselect.js    ← Plugin multiselect con tags (vanilla JS)
Instalación rápida

Agrega esto al <head> de tu layout compartido (application.html.erb).

application.html.erb — <head>
<!-- Fuentes -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet">

<!-- Estilos del sistema (en orden) -->
<link rel="stylesheet" href="styles.css">
<link rel="stylesheet" href="animations.css">
<link rel="stylesheet" href="components.css">

<!-- JS del multiselect (al final del body o con defer) -->
<script src="multiselect.js" defer></script>
⚠️ Carga styles.css antes que components.css — el segundo usa variables definidas en el primero.

Multiselect con tags

El plugin multiselect.js convierte un <select multiple> en un campo con chips. Las selecciones quedan visibles como tags dentro del input.

Puedes buscar escribiendo en el campo. Haz clic en × para quitar.
HTML — auto-inicialización con data-multiselect
<!-- Se inicializa solo al cargar la página -->
<select
  multiple
  data-multiselect
  data-placeholder="Seleccionar opciones…"
  data-max="5"
  name="contratistas[]"
>
  <option value="1">Manuel Rodríguez</option>
  <option value="2" selected>Alejandro Torres</option>
</select>

<!-- O inicializar manualmente por JS -->
<script>
  initMultiselect('#mi-select', { placeholder: 'Elegir…', max: 3 });
</script>
API JavaScript del multiselect
// Leer valores seleccionados
document.querySelector('#mi-select').msGetValues();
// → ['1', '3', '5']

// Limpiar todo
document.querySelector('#mi-select').msClearAll();

// Seleccionar todo
document.querySelector('#mi-select').msSelectAll();

// Establecer valores programáticamente
document.querySelector('#mi-select').msSetValues(['1','2']);

// Escuchar cambios (igual que un select normal)
document.querySelector('#mi-select').addEventListener('change', e => {
  console.log(e.target.msGetValues());
});
Atributos data- disponibles
AtributoDefaultDescripción
data-multiselect—Marca el select para auto-inicializar
data-placeholder"Seleccionar opciones…"Texto cuando no hay nada seleccionado
data-max0 (sin límite)Máximo de opciones seleccionables
data-searchable"true"false = deshabilita la búsqueda
ℹ️ El <select> original permanece oculto pero funcional. Sus valores se actualizan en tiempo real → compatible con formularios Rails (name="campo[]") y con form.submit().

Validación y estados de error

Agrega .is-error o .is-success al input. Muestra mensajes con .form-error-msg.

Formato de RFC inválido
Correo verificado
HTML + JS (Rails)
<!-- HTML con clase de estado -->
<input class="form-input is-error" value="ROCM123">
<div class="form-error-msg">Formato de RFC inválido</div>

# ERB con errores de modelo Rails
<% if @contratista.errors[:rfc].any? %>
  <input class="form-input is-error" value="<%= @contratista.rfc %>">
  <div class="form-error-msg"><%= @contratista.errors[:rfc].first %></div>
<% else %>
  <input class="form-input is-success" value="<%= @contratista.rfc %>">
<% end %>

Progreso circular

Para KPI compactos en tarjetas o dashboards. Controlado con stroke-dashoffset.

80%
Completas
40%
Pendientes
10%
Con error
HTML + fórmula de dashoffset
<div class="ring-progress">
  <svg viewBox="0 0 44 44" width="48" height="48">
    <circle class="ring-track" cx="22" cy="22" r="20"/>
    <circle class="ring-fill rf-success" cx="22" cy="22" r="20"
            style="stroke-dashoffset: 25"/>  <!-- 80% -->
  </svg>
  <span class="ring-label">80%</span>
</div>

# Fórmula ERB: dashoffset = 126 * (1 - porcentaje/100)
style="stroke-dashoffset: <%= (126 * (1 - @pct / 100.0)).round(1) %>"

# Variantes de color: rf-success | rf-warning | rf-danger | (azul por default)

Navbar completo

Fondo #13151f, texto cyan, dropdown blanco. Copia este JS al final de cada vista.

Estructura mínima
<nav class="navbar">
  <div class="navbar-left">
    <div class="nav-menu">
      <div class="nav-trigger">Procesos ▾</div>
      <div class="nav-dropdown">
        <a href="/procesos/carga">Carga Asistencia</a>
        <a href="/procesos/pre-factura" class="active">Pre Factura</a>
      </div>
    </div>
    <a class="nav-cerrar" href="/logout">Cerrar sesión</a>
  </div>
  <div class="navbar-right">
    <button class="btn-sync">↻ Sincronizar relojes</button>
  </div>
</nav>
<div class="sub-band"></div>

<!-- JS al final del body -->
<script>
document.querySelectorAll('.nav-menu').forEach(m => {
  m.querySelector('.nav-trigger').addEventListener('click', e => {
    e.stopPropagation();
    const open = m.classList.contains('open');
    document.querySelectorAll('.nav-menu.open').forEach(x => x.classList.remove('open'));
    if (!open) m.classList.add('open');
  });
});
document.addEventListener('click', () =>
  document.querySelectorAll('.nav-menu.open').forEach(m => m.classList.remove('open'))
);
</script>
